This repository contains "manual test cases" for the Swag Labs demo website, written in Excel.  
The test cases cover different functionalities, including login, logout, cart operations, checkout, sorting, sidebar, and footer links.

## Scope
- Login functionality
- Logout functionality
- Add / Remove items from cart
- Sorting products
- Checkout information
- Sidebar menu
- Footer links

## Test Cases
- TS_001: Verify login page
- TS_002: Add random item to cart
- TS_003: Add all items to cart
- TS_004: Sort items based on filters
- TS_005: Remove item from cart
- TS_006: Checkout information (First Name, Last Name, Zip Code)
- TS_007: Verify LinkedIn link in footer
- TS_008: Verify sidebar menu and reset app state
- TS_009: Verify logout functionality

## Tools & Skills
- Manual Testing
- Test Case Design
- Excel

## Bugs Found
- Some items were not added to the cart
- Items are not sorted correctly
- Last Name field mirrors First Name during checkout
- Reset App State in sidebar does not work as expected
